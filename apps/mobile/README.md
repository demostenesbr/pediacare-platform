# Mobile MVP (Expo)

Cliente mobile minimo para demonstrar consumo da API principal.

## Como executar

1. Instale dependencias:
   - `npm install`
2. Configure o endpoint da API:
   - `set EXPO_PUBLIC_API_URL=http://localhost:3000`
3. Inicie:
   - `npm run start`

A tela inicial lista os conteudos retornados por `GET /content`.
